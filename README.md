# Binary to Text Converter

This is a simple and elegant web-based tool to convert text to binary and binary to text in real-time. It's built with modern web technologies and features a clean, dark-mode interface. The application fully supports Unicode, including Thai characters.

## âœ¨ Features

*   **Real-time Conversion:** Instantly converts as you type.
*   **Bidirectional:** Converts from text to binary and binary to text.
*   **Unicode Support:** Handles a wide range of characters, including Thai.
*   **Clean & Modern UI:** Aesthetically pleasing dark-mode interface built with Tailwind CSS.
*   **Copy & Clear:** Easily copy the converted output or clear the text areas with a single click.
*   **Responsive Design:** Works seamlessly on both desktop and mobile devices.
*   **No Backend Required:** A purely client-side application.
